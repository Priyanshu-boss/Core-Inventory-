- Optimized production build

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/your-username/coreinventory.git
```

Navigate to project folder

```bash
cd coreinventory
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
